# ClimaTempo
Projeto de um aplicativo para demonstar a temperatura da região (por geolocalização) e dos próximos 7 dias, usando Xamarin.FORMS 4.4, com testes unitários usando o framework NUnit.
 
 # NuGet usados
  - Xamarin.Essentials
  - Newtonsoft.Json
  - NUnit (teste unitário)
